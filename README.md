# Ordering-system
Este projeto implementa o back-end de um sistema de realização de pedidos, desenvolvido em Java.
A aplicação é responsável pelo gerenciamento de usuários, produtos e pedidos, aplicando regras de negócio, validações, controle de autenticação e persistência de dados.
O sistema foi projetado seguindo boas práticas de arquitetura, organização de código e separação de responsabilidades, visando escalabilidade, manutenção e segurança.
